Build the project

Dependencies

The project requires:

  • Make ‚Äì sudo apt install make
  • CMake ‚Äì sudo apt install cmake
  • Git ‚Äì sudo apt install git
  • Qt5 (for the GUI) ‚Äì sudo apt install qtbase5-dev

Installing Dependencies on Windows

We recommend using MinGW. Install:

pacman -S mingw-w64-x86_64-make mingw-w64-x86_64-cmake mingw-w64-x86_64-git mingw-w64-x86_64-qt5

Also requires:

  • GCC 13+
  • C++23 support

Compilation

Build in release or debug mode:

make release
make debug

Running the Game

Launching the Program

Choose between two interfaces:

  • Graphical: ./royal-blocks-gui
  • Terminal: ./royal-blocks-tui

To start the server:

./royal-blocks-server

Choosing the IP and Port

You can define a port or let it use the environment variable SERVER_PORT (default: 1234).

Clients can modify IP and port in:

  • ~/.config/royal-blocks/config.json (Linux)
  • %APPDATA%\royal-blocks\config.json (Windows)

Terminal Display Tips

If the UI is cut off, zoom out in the terminal window.
